# EduTrack - Student Assessment System

EduTrack is a modern, browser-based dashboard for teachers to manage students, assessments, and grades.

## Features
-   **Student Management**: Add/Remove students with manual School IDs.
-   **Attendance**: Daily tracking with email alerts for students with >3 absences.
-   **Gradebook**: Automatic average calculation with support for Bonus marks.
-   **Responsive**: Works on Desktop, Tablet, and Mobile.
-   **Offline-First**: Uses LocalStorage; no backend server required.

## How to Run Locally
Simply open `index.html` in your web browser.

## How to Deploy to Vercel
Since EduTrack is a static site (HTML/CSS/JS), deployment is instant.

### Option 1: Drag & Drop
1.  Go to [Vercel.com](https://vercel.com) and log in.
2.  Click **"Add New Project"**.
3.  Upload the `web mini project` folder.
4.  Click **Deploy**.

### Option 2: Vercel CLI
If you have the Vercel CLI installed:
